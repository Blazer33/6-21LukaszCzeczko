1. W jaki sposób można określić właściwość danego obiektu?

Właściwość obiektu można określić na dwa sposoby:

  • Notacja kropkowa
let osoba = { imie: "Łukasz", wiek: 18 };
console.log(osoba.imie);
  • Notacja nawiasowa
let osoba = { imie: "Łukasz", wiek: 18 };
console.log(osoba["wiek"]);

2. W jaki sposób można odwołać się do pól i metod?

Do pól obiektu:

console.log(osoba.imie); 
console.log(osoba["wiek"]);

Do metod:

let osoba = {
    imie: "Łukasz",
    przedstawSie: function() {
        return "Cześć, mam na imię " + this.imie;
    }
};
console.log(osoba.przedstawSie());

3. Wymień cechy charakterystyczne konstruktora.

  • Jest funkcją służącą do tworzenia nowych obiektów.
  • Nazywany jest z wielkiej litery.
  • Używa this do przypisywania wartości polom obiektu.
  • Tworzenie obiektów odbywa się za pomocą new.

4. Konstruowanie obiektu na podstawie przykładu – obiekt ma trzy pola oraz jedną metodę.

let samochod = {
    marka: "Toyota",
    model: "Corolla",
    rok: 2022,
    pokazInformacje: function() {
        return this.marka + " " + this.model + " z roku " + this.rok;
    }
};
console.log(samochod.pokazInformacje());

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

this odnosi się do bieżącego obiektu i pozwala odwoływać się do jego pól i metod.

Pola oraz metody oddzielamy przecinkami w deklaracji obiektu:

let osoba = {
    imie: "Anna",
    wiek: 30,
    przedstawSie: function() {
        return "Mam na imię " + this.imie;
    }
};

6. Konstruktor w JS na podstawie przykładu wraz z opisem.

Konstruktor to specjalna funkcja służąca do tworzenia nowych obiektów.

function Samochod(marka, model, rok) {
    this.marka = marka;
    this.model = model;
    this.rok = rok;
}
let auto = new Samochod("Ford", "Focus", 2020);
console.log(auto.marka);

7. Znaczenie słowa kluczowego new wraz z przykładem.

new tworzy nowy obiekt i przypisuje this do niego.

function Osoba(imie, wiek) {
    this.imie = imie;
    this.wiek = wiek;
}
let osoba1 = new Osoba("Łukasz", 18);
console.log(osoba1.imie);

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem.

prototype pozwala dodawać metody i właściwości do wszystkich obiektów danego typu.

function Osoba(imie) {
    this.imie = imie;
}
Osoba.prototype.przedstawSie = function() {
    return "Cześć, jestem " + this.imie;
};
let osoba1 = new Osoba("Łukasz");
console.log(osoba1.przedstawSie());
